Test-Ideas List
	Cliente
		O usuário pode escolher o LookAndFeel que desejar e torná-lo o padrão do programa.

	Servidor
		Tempo de resposta entre cliente e servidor deve ser rápida (< 2 segundos)
		O servidor trata corretamente o comando enviado
		O servidor responde corretamente a solicitação do comando
		Exemplo de comando:
			Cliente envia comando "cadastro%login%email%senha"
				A senha poderia ser encriptada para maior segurança
			Servidor trata o comando e adiciona o cliente na base de dados
			Servidor retorna confirmação para cliente
